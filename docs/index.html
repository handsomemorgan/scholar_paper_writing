PaperForge 论文自动写作助手

基于 7-Agent 流水线的学术论文自动写作系统。
输入需求，AI 自动完成文献检索、写作、校验的全流程。

快速开始 查看源码

界面预览

以下为实际 Web GUI 的操作界面。支持切换 DeepSeek / Claude / GPT 模型，以及五款背景主题。

在本地运行 PaperForge

此功能需要在本地启动后端服务。请克隆仓库并运行：

git clone https://github.com/handsomemorgan/scholar_paper_writing.git
cd scholar_paper_writing
pip install -r requirements.txt
cp .env.example .env # 配置 API Key
python web_server.py

然后访问 http://127.0.0.1:5000 即可使用完整功能。

核心特性

7-Agent 流水线

分类 → 模板检索 → 关键词提取 → 文献检索 → 文献分析 → 论文撰写 → 格式校验，全自动端到端。

多模型支持

DeepSeek、Anthropic Claude、OpenAI GPT 自由切换，自动检测已配置的 API Key。

学术文献检索

通过 arXiv 公开 API 检索真实学术文献，三级回退容错链，确保文献数量充足。

格式模板管理

RAG 向量检索匹配 5 大类学术格式模板，理科/文科/工科/实验报告/调研报告。

中英双语输出

自动生成中文论文 + 英文版本，满足国内外期刊投稿需求。

实时进度追踪

Web 界面 SSE 流式推送 8 个阶段的实时进度，完整掌握生成流程。

7-Agent 流水线架构

1 论文分类
→
2 模板检索

快速开始

  1. 克隆仓库

    将项目克隆到本地

    git clone https://github.com/handsomemorgan/scholar_paper_writing.git
cd scholar_paper_writing

  2. 安装依赖

    创建虚拟环境并安装依赖包

    python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

  3. 配置 API Key

    复制环境变量模板，填入你的 API Key

    cp .env.example .env
# 编辑 .env 填入 DEEPSEEK_API_KEY

  4. 启动 Web 界面

    启动后浏览器访问 http://127.0.0.1:5000

    python web_server.py
